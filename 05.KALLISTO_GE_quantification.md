I used KALLISTO to firstly index the Artemia sinica transcriptome. I then use it now to map the raw RNAseq reads (Christelle's (2022) and mine(2025)) to this indexed transcriptome.

# 1. Map the 2025 F2BC RNAseq samples 

Reads location : `/nfs/scistore18/vicosgrp/clasne/SinKaz_hybrid_GE/raw_files_GE_SINKAZ_2025/raw_FASTQ_files_only_2025/`

Script: `KALLISTO_reads_mapping.sh`

```ruby
# load module
module load kallisto

# loop
for base in $(ls /nfs/scistore18/vicosgrp/clasne/SinKaz_hybrid_GE/raw_files_GE_SINKAZ_2025/raw_FASTQ_files_only_2025/*1_001.fastq.gz | sed -r 's/1_001.fastq.gz//' | uniq)
do
srun kallisto quant -t 16 -i /nfs/scistore18/vicosgrp/clasne/SinKaz_hybrid_GE/transcriptome_files/Indexed_Asinica500_transcriptome.idx -o "${base}_KALLISTOmapped_25" -b 100 "${base}1_001.fastq.gz" "${base}2_001.fastq.gz"
done
```
