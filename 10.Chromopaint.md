# Chromopaint script adapatation

Beatriz's script for chromopaint can be found here : https://git.ist.ac.at/bvicoso/zsexasex2021/-/blob/master/scripts/chromopaint.pl. Make a `script_chromopaint.pl` with pico and copy and paste in it the content of the original chromopaint script. IMPORTANT NOTE the chromopaint script was modified so that genes with only 1 SNP were accepted in the analysis. The following command line in Beatriz's script `if (($num0{$sample}+$num1{$sample})>10` ) was replaced by `if (($num0{$sample}+$num1{$sample})>0` )

## Chromopaint on 2025 F2BC SM, SF, KM, KF subsets

Script: `chromopaint_BCs_2025_script.sh`

```ruby
#command
perl script_chromopaint.pl SinKaz_filtered_FIXED_SNP_subset_BCs_ALL_2025_nomissing.vcf
perl script_chromopaint.pl SinKaz_filtered_FIXED_SNP_subset_BC_KF_2025_nomissing.vcf
perl script_chromopaint.pl SinKaz_filtered_FIXED_SNP_subset_BC_KM_2025_nomissing.vcf
perl script_chromopaint.pl SinKaz_filtered_FIXED_SNP_subset_BC_SF_2025_nomissing.vcf
perl script_chromopaint.pl SinKaz_filtered_FIXED_SNP_subset_BC_SM_2025_nomissing.vcf
```
