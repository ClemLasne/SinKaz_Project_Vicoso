# 2. SNP calling

This step used different programs and functions:

**samtools**
- `faidx` function indexes reference sequence in the FASTA format or extract subsequence from indexed reference sequence. If no region is specified, faidx will index the file and create <ref.fasta>.fai 


**BCFtools:** BCFtools is a set of utilities that manipulate variant calls in the Variant Call Format (VCF) and its binary counterpart BCF. All commands work transparently with both VCFs and BCFs, both uncompressed and BGZF-compressed.

**`bcftools mpileup` function** : generates genotype likelihoods at each genomic position with coverage
- `-a` : Comma-separated list of FORMAT and INFO tags to output:
  
<img width="995" alt="Screenshot 2023-10-13 at 17 01 50" src="https://github.com/ClemLasne/SinKaz_project/assets/24788785/acf5a3ce-f73b-4bba-93a6-998a1b426e1f">

- `-Ou`: to work with uncompressed BCF output
  
- `-f` : The faidx-indexed reference file in the FASTA format

  
**`bcftools call` function** : makes the actual calls
- `-v` : asks to output only variant sites
- `-o` : selects the output format
- `-f` : comma-separated list of FORMAT fields to output for each sample. Currently GQ and GP fields are supported.
- `-mO z` : alternative model for multiallelic and rare-variant calling (m) output type (O) compressed VCF (z),

**VCFtools:** provides easily accessible methods for working with complex genetic variation data in the form of VCF files. This toolset can be used to perform the following operations on VCF files: Filter out specific variants, Compare files, Summarize variants, Convert to different file types, Validate and merge file, Create intersections and subsets of variants.

- `--gzvcf FILE` : read a compressed (gzipped) VCF input file
- `--recode` : Output a new vcf file from the input vcf file
- `--remove-indels` : removes any indel sites (from the output file)
- `--maf` (minor allele frequency) : Include only sites with a Minor Allele Frequency greater than or equal to the "--maf" value.
- `--max-missing` : Exclude sites on the basis of the proportion of missing data (defined to be between 0 and 1, where 0 allows sites that are completely missing and 1 indicates no missing data allowed).
- `--minQ`: Includes only sites with Quality value above this threshold.
- `--min-meanDP` and `--max-meanDP`: Includes only sites with mean depth values (over all included individuals) greater than or equal to the "--min-meanDP" value and less than or equal to the "--max-meanDP" value.
- `--minDP` and `--maxDP`: Includes only genotypes greater than or equal to the "--minDP" value and less than or equal to the "--maxDP" value.
- `--stdout` : directs the vcftools output to standard out so it can be piped into another program or written directly to a filename of choice.

## SNP calling for the 2022 F2BC samples (Christelle's RNA seq samples)

**IMPORTANT** : Here I only use the 2025 RNAseq parental sinica and kaz samples to get the alternatively fixed SNPs between the 2 parental populations, not the "old" RNA seq samples.

Script : `SNP_calling_VCF.sh`

Content of the script:
```ruby
#load any module you need here
module load bwa
module load samtools
module load bcftools
module load vcftools

#commands to run

## index transcriptome
srun samtools faidx /nfs/scistore18/vicosgrp/clasne/SinKaz_hybrid_GE/transcriptome_files/Asinica500.fa

# actual SNP calling step
srun bcftools mpileup --threads 50 -a AD,DP,SP -Ou -f /nfs/scistore18/vicosgrp/clasne/SinKaz_hybrid_GE/transcriptome_files/Asinica500.fa BC_SM_F1.bam BC_SM_F10.bam BC_SM_F2.bam BC_SM_F3.bam BC_SM_F4.bam BC_SM_F5.bam BC_SM_F6.bam BC_SM_F7.bam BC_SM_F8.bam BC_SM_F9.bam BC_SM_M1.bam BC_SM_M10.bam BC_SM_M2.bam BC_SM_M3.bam BC_SM_M4.bam BC_SM_M5.bam BC_SM_M6.bam BC_SM_M7.bam BC_SM_M8.bam BC_SM_M9.bam 25_535_SIN_F.bam 25_536_SIN_F.bam 25_537_SIN_M.bam 25_538_SIN_M.bam 25_539_KAZ_F.bam 25_540_KAZ_F.bam 25_541_KAZ_M.bam 25_542_KAZ_M.bam | bcftools call -v -f GQ,GP -mO z -o SinKaz_2022_samples_.vcf.gz --threads 50

# output filtering steps for quality and coverage
srun vcftools --gzvcf SinKaz_2022_samples_.vcf.gz --remove-indels --maf 0.1 --max-missing 0.9 --minQ 30 --min-meanDP 4 --max-meanDP 200 --minDP 4 --maxDP 200 --recode --stdout > SinKaz_2022_samples_filtered.vcf
```
