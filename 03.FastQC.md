# 1. Run FastQC on fastq files

### The following pipeline only applied to Clem's 2025 RNA seq samples

Directory where the fastq.gz files are: `/nfs/scistore18/vicosgrp/clasne/SinKaz_hybrid_GE/raw_files_GE_SINKAZ_2025/raw_FASTQ_files_only_2025`

Script: 
```ruby
pico FASTQC_script.sh
```

```ruby
#!/bin/bash
#SBATCH --partition=defaultp
#SBATCH --job-name=fastqc
#SBATCH -c 5
#SBATCH -t 48:00:00
#SBATCH --mail-user=clementine.lasne@ist.ac.at
#SBATCH --mail-type=ALL
#SBATCH --no-requeue
#SBATCH --ntasks=1
#SBATCH -N 1
#SBATCH --mem=8000
#SBATCH --output=outcome_fastqc_2025.out
#SBATCH --export=NONE
unset SLURM_EXPORT_ENV

#Set the number of threads to the SLURM internal variable

export OMP_NUM_THREADS=$SLURM_CPUS_PER_TASK

#load modules
module load java
module load fastqc

#command
srun fastqc /nfs/scistore18/vicosgrp/clasne/SinKaz_hybrid_GE/raw_files_GE_SINKAZ_2025/raw_FASTQ_files_only_2025/*.fastq.gz
```
