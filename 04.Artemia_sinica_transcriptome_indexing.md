# Copy A. sinica transcriptome

Location of the sinica transcriptome `/nfs/scistore18/vicosgrp/bvicoso/Artemia_expression_Oct2019/1-Transcriptome/Asinica500.fa`

Make a directory to host a copy of the sinica transcriptome and copy it in there:

```ruby
cp /nfs/scistore18/vicosgrp/bvicoso/Artemia_expression_Oct2019/1-Transcriptome/Asinica500.fa /nfs/scistore18/vicosgrp/clasne/SinKaz_hybrid_GE/transcriptome_files/ 
```

*Note that this directory (`/nfs/scistore18/vicosgrp/bvicoso/Artemia_expression_Oct2019/1-Transcriptome`)also contains a `Asinica500.fa.aa` file*

# Transcriptome indexing with Kallisto

```ruby
pico transcriptome_indexing_Kallisto_script.sh
```
Content of the script:
```ruby

#load any module you need here
module load kallisto

#run commands on SLURM's srun
srun kallisto index -i Indexed_Asinica500_transcriptome.idx /nfs/scistore18/vicosgrp/clasne/SinKaz_hybrid_GE/transcriptome_files/Asinica500.fa
```

Run the script:
```ruby
sbatch transcriptome_indexing_Kallisto_script.sh
```
The file **Indexed_Asinica500_transcriptome.idx** was produced.
