Use the `ExpressionSummary_SINKAZ_01092025.txt` produced in the previous step by Sleuth and normalise gene expression across samples with the R package NormaliserDE.

# 3. Normalisation GE 

```ruby
pico NormalizerDE_Clem.R
```

```ruby
# open gene expression dataset to be normalised 
expf <- read.table("ExpressionSummary_SINKAZ_01092025.txt", head=T, sep="")
# delete the first column if this column is gene names
expf_1 <- expf[, -1]
# create a new matrix that has a number of columns and rows that match the expression gene dataset
bolFMat_1<-as.matrix(expf_1, nrow = nrow(expf_1), ncol = ncol(expf_1))

# load the data normalisation package
library(NormalyzerDE)

# run data normalisation
expf_2<-performQuantileNormalization(bolFMat_1, noLogTransform = T)

# save the matrix containing normalised data as a dataframe
expf_2<-as.data.frame(expf_2)

# reintroduce the gene name column into the dataframe
expf_2 <- data.frame(expf[, 1], expf_2)
# rename the first column of the new dataframe (normalised data)
colnames(expf_2)[1] <- "gene"
# check that the gene names match between expf and expf_2
head(expf)
head(expf_2)
# save the new dataset of normalised data
write.table(expf_2, file = "Normalized_ExpressionSummary_SINKAZ_01092025.txt", quote=F)
```
